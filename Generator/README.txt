Run with
    java -jar target/Generator-0.0.1-SNAPSHOT-jar-with-dependencies.jar

Demonstrates
    Spring
        Scope("prototype") - using component-scan and using annotation-config
    Maven
        Use of specific Java version (maven defaults to 1.5)
    Java
        Use of generics (i.e. List<E>)
        Use of unbounded wildcard types (i.e. List<?>)
        Use of Varargs
        Use of method invocation through reflection api
